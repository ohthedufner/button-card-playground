Presets
Five boolean properties control every visible element independently. Defaults: show_name and show_icon are true; everything else is false. The card still occupies space when all are hidden — use hidden: true to remove it from layout entirely.
show_name show_icon show_state show_label show_last_changed
show_name
Entity friendly name below the icon
show_icon
MDI icon centered in the card
show_state
Current state text (on / off / numeric)
show_label
Secondary text line below state
show_last_changed
Elapsed time since last state change
